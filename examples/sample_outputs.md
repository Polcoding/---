# 모범 출력 예시

## 예시 1. 지역관서별 순찰차 탑승물품 현황 조사 지시 공문

# 업무 해석 요약

지역관서별 순찰차 탑승물품 현황을 파악하기 위한 조사 지시 공문, 조사표 양식, 작성요령 초안이 필요합니다. 실제 관서명, 차량번호, 세부 장비 수량은 포함하지 않습니다.

# 확인 필요 사항

- 수신처: `[수신처 확인 필요]`
- 조사기준일: `[확인 필요]`
- 제출기한: `[확인 필요]`
- 제출방법: `[확인 필요]`
- 담당부서: `[담당부서 확인 필요]`

# 보안ㆍ개인정보 점검

- 실제 관서명은 사용하지 않았습니다.
- 차량번호는 사용하지 않았습니다.
- 세부 장비 수량은 작성하지 않았습니다.
- 모든 운영정보는 placeholder로 표시했습니다.

# 공문 초안

```text
문서번호: [문서번호 확인 필요]
시행일자: [YYYY.MM.DD. 확인 필요]
수    신: [수신처 확인 필요]
참    조: [참조 확인 필요]
제    목: 순찰차 탑승물품 현황 조사 요청

1. [업무 목적 확인 필요]을 위하여 각 대상 기관의 순찰차 탑승물품 현황을 확인하고자 합니다.

2. 이에 각 대상 기관에서는 아래 사항을 확인하여 기한 내 제출하여 주시기 바랍니다.

  가. 조사대상: [대상 차량 범위 확인 필요]
  나. 조사기준일: [확인 필요]
  다. 조사항목: 탑승물품 구분, 보유 여부, 관리 상태, 비고
  라. 제출기한: [확인 필요]
  마. 제출방법: [확인 필요]
  바. 작성 유의사항: 차량번호, 개인 식별정보, 세부 내부 운영정보는 기재하지 않음

3. 제출된 자료는 현황 파악과 관리 개선 검토를 위한 기초자료로 활용될 예정이므로 정확한 자료가 제출될 수 있도록 협조하여 주시기 바랍니다.

붙임  1. 순찰차 탑승물품 현황 조사표 1부.  끝.
```

# 조사표 양식

| 구분 | 탑승물품 항목 | 보유 여부 | 관리 상태 | 비고 |
|---|---|---|---|---|
| [기관 A] | [물품 항목 확인 필요] | [확인 필요] | [확인 필요] | [확인 필요] |
| [기관 B] | [물품 항목 확인 필요] | [확인 필요] | [확인 필요] | [확인 필요] |

# 작성요령

- 실제 관서명 대신 `[기관 A]`, `[기관 B]` 형식을 사용합니다.
- 차량번호와 개인 식별정보는 작성하지 않습니다.
- 세부 수량이 민감한 운영정보에 해당할 수 있으면 `[확인 필요]`로 표시합니다.
- 제출 전 내부 승인 필요 여부를 확인합니다.

# 발송 전 체크리스트

- 수신처 확인
- 조사기준일 확인
- 제출기한 확인
- 개인정보 포함 여부 확인
- 차량 식별정보 포함 여부 확인
- 첨부 조사표 확인

## 예시 2. 행사 음향장비 임차 견적 요청 메일

# 업무 해석 요약

행사 운영에 필요한 음향장비 임차 가능 여부와 견적 정보를 사전 검토하기 위한 업체 연락 메일 초안입니다. 계약 또는 업체 선정이 확정된 표현은 사용하지 않습니다.

# 확인 필요 사항

- 행사명: `[행사명 확인 필요]`
- 행사일시: `[일시 확인 필요]`
- 장소: `[장소 확인 필요]`
- 필요 장비: `[장비 목록 확인 필요]`
- 회신기한: `[확인 필요]`

# 업체 메일 초안

```text
제목: [행사명 확인 필요] 음향장비 임차 사전 견적 요청

안녕하세요.
[기관 A]의 [담당자 A]입니다.

우리 기관에서는 [행사명 확인 필요] 운영을 검토 중이며, 행사 진행에 필요한 음향장비 임차 가능 여부와 사전 견적을 확인하고자 합니다.

아래 내용을 참고하시어 가능 여부와 견적 정보를 회신하여 주시면 감사하겠습니다.

1. 행사명: [행사명 확인 필요]
2. 행사일시: [일시 확인 필요]
3. 행사장소: [장소 확인 필요]
4. 요청사항: [장비 목록 확인 필요]
5. 회신기한: [확인 필요]

본 요청은 사전 검토를 위한 견적 요청이며, 계약 또는 업체 선정이 확정된 사항은 아닙니다. 회신 자료에는 개인정보 또는 민감정보가 포함되지 않도록 유의하여 주시기 바랍니다.

감사합니다.
```

# 계약 미확정 문구

`본 요청은 사전 검토를 위한 자료 요청이며, 계약 또는 업체 선정이 확정된 사항은 아닙니다.`

# 발송 전 체크리스트

- 업체 수신 주소 확인
- 행사 정보 확인
- 계약 미확정 문구 포함 여부 확인
- 개인정보 또는 민감정보 포함 여부 확인
- 첨부파일 포함 여부 확인
