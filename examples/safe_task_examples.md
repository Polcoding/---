# 안전한 업무 지시 예시

## 목적

AI가 처리 가능한 안전한 업무 지시 예시를 정리합니다. 모든 예시는 가상 정보 또는 placeholder만 사용합니다.

## 공문 초안

```text
[기관 A]에 [점검 대상] 현황 제출을 요청하는 공문 초안을 작성해줘. 실제 기관명, 문서번호, 수량은 모두 [확인 필요]로 표시해줘.
```

## 기획서 초안

```text
[사업명 확인 필요] 추진계획서 초안을 작성해줘. 대상, 일정, 예산은 아직 확정되지 않았으니 확인 필요로 표시해줘.
```

## 업체 메일 초안

```text
[업체 A]에 [물품 또는 용역명 확인 필요] 사전 견적을 요청하는 메일 초안을 작성해줘. 계약이나 선정이 확정된 것처럼 쓰지 말아줘.
```

## 조사표 양식

```text
[대상 기관]별 [점검 대상] 현황을 조사하기 위한 표 양식을 만들어줘. 실제 수량과 기관명은 넣지 말고 항목명과 작성요령만 만들어줘.
```

## 작성요령

```text
[문서 유형]을 작성할 때 필요한 항목과 작성요령을 정리해줘. 실제 사례 대신 placeholder를 사용해줘.
```

## 자료조사 요청 정리

```text
[자료명 확인 필요] 제출 요청 문안을 만들어줘. 제출기한, 제출방법, 담당자는 모두 확인 필요로 표시해줘.
```

## 회의자료 정리

```text
[회의 주제 확인 필요] 회의 메모를 검토자료 형식으로 정리해줘. 참석자 이름은 쓰지 말고 역할명으로 표시해줘.
```
