# 비식별 샘플 템플릿

## 사용 원칙

이 템플릿에는 원문 내용을 직접 붙여넣지 않습니다. 원문 요약도 작성하지 않습니다. 기존 문서에서 문장 구조, 문체, 번호 체계, 서식 패턴만 추출하여 placeholder 기반 샘플로 새로 작성합니다.

## 원문 사용 금지 확인

- [ ] 원문 문장을 직접 붙여넣지 않음
- [ ] 원문 내용을 요약하지 않음
- [ ] 실제 개인정보를 포함하지 않음
- [ ] 실제 기관명ㆍ관서명ㆍ업체명을 포함하지 않음
- [ ] 실제 차량번호ㆍ민원번호ㆍ접수번호를 포함하지 않음
- [ ] 실제 내부 운영정보를 포함하지 않음
