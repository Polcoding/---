# 비식별 문체 샘플 작성 템플릿

## 작성 원칙

- 실제 원문 문장을 그대로 옮기지 않습니다.
- 실제 파일명, 문서명, 기관명, 담당자명, 업체명, 날짜, 문서번호를 기록하지 않습니다.
- 실제 개인정보, 민감정보, 내부 운영정보를 기록하지 않습니다.
- 문장 구조, 번호 체계, 목차 구조, 붙임 표기, 표 구성 등만 일반화합니다.
- 비식별 샘플에는 placeholder를 사용합니다.
- 샘플 작성 전 보안등급이 A 또는 B 후보인지 확인합니다.
- C등급은 내부 참고만 가능하므로 샘플 후보에서 제외합니다.
- D등급은 외부 처리 금지 대상으로 분류하고 샘플 작성 대상에서 제외합니다.
- Custom GPT Knowledge 업로드는 아직 하지 않습니다.

## 샘플 기본 양식

```markdown
## [샘플 번호]. [문서 유형] 문체 샘플

샘플 번호:
[SAMPLE-001]

작성일:
[작성일 placeholder]

문서 유형:
[문서 유형]

관련 후보코드 또는 문서코드:
[후보코드 placeholder]

보안등급:
[A등급 후보/B등급 후보]

원문 사용 여부:
원문 미사용. 실제 문장을 옮기지 않고 문장 구조만 일반화한 예시.

추출 대상:
- [문체 패턴]
- [번호 체계]
- [목차 구조]
- [붙임 표기]
- [표 구성]

원문에서 가져오지 않는 정보:
- 실제 파일명
- 실제 문서명
- 실제 기관명, 관서명, 업체명, 담당자명
- 실제 날짜, 문서번호, 민원번호, 차량번호
- 실제 장비명, 실제 수량, 실제 견적 금액
- 실제 개인정보, 민감정보, 내부 운영정보

비식별 문체 샘플:
[업무 발생 배경]에 따라 [대상 기관/부서]의 [조사 대상] 현황을 확인하고, 향후 [활용 목적]을 위한 기초자료로 활용하고자 합니다.

각 [대상 기관/부서]에서는 [조사 기준일] 현재의 [조사 대상] 현황을 붙임 서식에 따라 작성하여 [제출기한]까지 [제출방법]으로 제출하여 주시기 바랍니다.

사용 가능한 상황:
- 문체와 구조만 참고할 때
- placeholder 기반 샘플로 누적할 때
- A/B등급 후보 문서의 비식별 샘플 검수 전 초안을 작성할 때

금지되는 활용:
- 실제 원문 학습
- 실제 원문 요약
- C/D등급 문서 샘플화
- 실제 개인정보 또는 내부 운영정보 입력
- 자동화 구현 또는 외부 연동

검수 체크리스트:
- [ ] 원문 문장을 그대로 옮기지 않았는가
- [ ] 원문 내용을 요약하지 않았는가
- [ ] 실제 파일명과 실제 문서명이 없는가
- [ ] 실제 기관명, 관서명, 업체명, 담당자명이 없는가
- [ ] 실제 날짜, 문서번호, 민원번호, 차량번호가 없는가
- [ ] 실제 장비명, 수량, 견적 금액이 없는가
- [ ] 모든 식별정보가 placeholder로 치환되었는가
- [ ] A 또는 B등급 후보로만 검토되었는가

검수 결과:
[통과/수정 필요/제외]

비고:
[비고 placeholder]
```

## 샘플 번호 규칙

- SAMPLE-001부터 순차 부여합니다.
- 문서유형별 접두어를 붙일 수도 있습니다.
  - 공문: OFFICIAL-SAMPLE-001
  - 기획서: PLAN-SAMPLE-001
  - 보고서: REPORT-SAMPLE-001
  - 업체메일: VENDOR-MAIL-SAMPLE-001
  - 조사표: SURVEY-SAMPLE-001

## placeholder 규칙

placeholder는 실제 기관명, 실제 관서명, 실제 업체명, 실제 사람 이름처럼 보이면 안 됩니다.

사용 가능한 placeholder 예시는 다음과 같습니다.

- [업무 발생 배경]
- [대상 기관/부서]
- [조사 대상]
- [조사 기준일]
- [제출기한]
- [제출방법]
- [담당부서]
- [공용 연락처]
- [활용 목적]
- [사업명]
- [행사명]
- [추진기간]
- [소요예산 확인 필요]
- [업체 A]
- [자료 요청 항목]
- [비식별 처리된 민원 요지]

## 작성 전 확인

- 원문을 열람하더라도 이 템플릿에는 원문 문장을 옮기지 않습니다.
- 실제 문서의 요지를 정리하지 않고, 문체와 구조만 새 문장으로 일반화합니다.
- C등급 또는 D등급으로 판단되는 문서는 이 템플릿으로 샘플을 작성하지 않습니다.
