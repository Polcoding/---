# XLSX 템플릿 폴더 안내

## 목적

이 폴더는 향후 XLSX 조사표 템플릿 파일을 보관하기 위한 위치입니다.

현재 단계에서는 실제 XLSX 파일을 보관하지 않습니다.

## 보관 가능 파일

향후 보관 가능한 파일 유형:

- 비식별 조사표 템플릿
- placeholder 기반 XLSX 템플릿
- 실제 개인정보나 내부 운영정보가 없는 테스트용 템플릿

## 보관 금지 파일

다음 파일은 이 폴더에 보관하지 않습니다.

- 실제 조사자료
- 실제 관서별 현황자료
- 실제 차량번호 또는 장비현황 포함 파일
- 민원자료
- 개인정보 포함 명단
- 업체 견적자료
- 대외비 또는 비공개 문서
- 내부망 자료
- 실제 업무 원본 파일

## placeholder 원칙

- 모든 템플릿에는 실제 값 대신 placeholder를 사용합니다.
- 예: {{title}}, {{standard_date}}, {{rows}}, {{checklist}}
- 실제 기관명, 담당자명, 연락처, 차량번호, 장비명, 수량은 사용하지 않습니다.

## 현재 상태

- 현재 단계: 설계 단계
- 실제 XLSX 파일: 없음
- 렌더링 코드: 없음
- 향후 Step에서 PoC 구현 여부 검토
