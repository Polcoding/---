# 문서 패턴 라이브러리

## 목적

자주 쓰는 행정문서 문장 패턴을 placeholder 기반으로 정리합니다. 이 문서는 실제 사례 모음이 아니라 문체ㆍ구조 패턴 라이브러리입니다.

## 자주 쓰는 공문 문장 패턴

```text
[업무 목적]을 위하여 [대상]에 대한 [요청 내용]을 추진하고자 합니다.
```

```text
관련 업무의 원활한 추진을 위하여 아래 사항을 확인하여 주시기 바랍니다.
```

## 조사 요청 문장

```text
[조사 목적]을 위하여 [대상 기관]의 [조사 항목] 현황을 조사하고자 합니다.
```

```text
조사 항목 중 내부 기준 확인이 필요한 사항은 `[내부 기준에 따라 확인 필요]`로 표시하여 주시기 바랍니다.
```

## 자료 제출 요청 문장

```text
각 대상 기관에서는 [제출자료 확인 필요]를 [제출기한 확인 필요]까지 제출하여 주시기 바랍니다.
```

```text
제출자료에는 개인정보, 민감정보, 내부 운영정보가 포함되지 않도록 확인하여 주시기 바랍니다.
```

## 협조 요청 문장

```text
업무 추진에 필요한 사항이 원활히 확인될 수 있도록 협조하여 주시기 바랍니다.
```

```text
세부 확인이 필요한 사항은 [담당부서 확인 필요]와 사전 협의하여 주시기 바랍니다.
```

## 업체 견적 요청 문장

```text
[업무명 확인 필요] 관련 사전 검토를 위하여 자료/견적 제출 가능 여부를 확인하고자 합니다.
```

```text
본 요청은 사전 검토를 위한 자료 요청이며, 계약 또는 업체 선정이 확정된 사항은 아닙니다.
```

## 추진배경 문장

```text
[업무 환경 변화]에 따라 [대상 업무]의 체계적인 추진 필요성이 제기되었습니다.
```

```text
[대상]의 이용 편의와 운영 효율을 높이기 위하여 [사업명 확인 필요] 추진을 검토하고자 합니다.
```

## 기대효과 문장

```text
본 계획을 통해 [대상]의 접근성을 높이고, 관련 업무의 일관성과 효율성을 강화할 수 있을 것으로 기대됩니다.
```

```text
향후 [후속 조치 확인 필요]를 위한 기초자료로 활용할 수 있습니다.
```

## 확인 필요 문장

```text
세부 일정, 예산, 대상, 담당부서는 `[확인 필요]`로 표시합니다.
```

```text
내부 기준에 따라 기재 범위와 공개 가능 여부를 확인해야 합니다.
```

## 금지 표현 목록

- 법적으로 문제 없습니다.
- 완전히 안전합니다.
- 업체 선정이 확정되었습니다.
- 계약 체결이 확정되었습니다.
- 예산이 확보되었습니다.
- 기존 공문 원문을 학습시킵니다.
- 내부자료를 외부 AI에 입력합니다.
- 민감정보를 포함하여 업체에 전달합니다.
