# HWPX 기관 표준 서식값 확인 기준

## 목적

이 문서는 HWPX 자동화 결과물이 공공기관 문서로 실무 검토 가능한 수준이 되도록, 기관 표준 서식값을 확인하고 기록하기 위한 기준을 정리합니다.

이번 단계에서는 실제 기관 HWPX 양식 원본을 저장소에 추가하지 않습니다.

확인되지 않은 서식값은 모두 [확인 필요]로 표시합니다.

## 기본 원칙

- 실제 기관 HWPX 양식 원본을 Git에 추가하지 않습니다.
- 실제 공문 원문을 사용하지 않습니다.
- 실제 개인정보, 기관명, 문서번호를 기록하지 않습니다.
- 폰트 파일은 저장소에 추가하지 않습니다.
- 서식값은 사람이 내부 기준을 확인한 뒤 수동으로 기록합니다.
- 확인 전에는 값을 추측하지 않고 [확인 필요]로 둡니다.
- HWPX 템플릿과 렌더러는 확인된 style_profile 기준을 따릅니다.

## 문서유형별 style profile

다음 문서유형별 style profile 후보를 정리합니다.

| style_profile_id | 문서유형 | 적용 대상 | 확인 상태 |
|---|---|---|---|
| style_official_letter_basic | 일반 공문 | 제목, 본문, 붙임 | 확인 필요 |
| style_survey_request_letter | 현황조사 지시 공문 | 공문 본문, 조사표 붙임 | 확인 필요 |
| style_project_plan_basic | 추진계획서 | 목차, 표, 일정, 예산 | 확인 필요 |
| style_result_report_basic | 결과보고서 | 결과 표, 향후계획 | 확인 필요 |
| style_review_report_basic | 검토보고서 | 검토의견, 위험요소 | 확인 필요 |

## style profile 필드

각 style profile은 다음 항목을 가질 수 있습니다.

- style_profile_id
- document_type
- title_font
- title_size
- title_align
- body_font
- body_size
- body_letter_spacing
- body_line_spacing
- body_paragraph_spacing
- body_indent
- table_border
- table_cell_margin
- table_header_style
- attachment_style
- closing_style
- page_margin
- review_status

모르는 값은 [확인 필요]로 유지합니다.

## 기록 시 주의사항

- 실제 기관명은 기록하지 않습니다.
- 실제 문서 원문은 기록하지 않습니다.
- 폰트 파일은 저장소에 추가하지 않습니다.
- 내부 승인 없이 실제 기관 양식을 Git에 추가하지 않습니다.
- 서식값은 확인 가능한 범위에서만 기록합니다.

## Step 30 결론

Step 30은 HWPX 자동화 결과물의 서식 품질을 높이기 위해 기관 표준 서식값 확인 항목을 정리하는 단계입니다.

현재 단계에서는 실제 서식값을 임의로 확정하지 않습니다.

확인되지 않은 값은 [확인 필요]로 유지합니다.

다음 단계에서는 확인된 style_profile을 기준으로 HWPX 템플릿 또는 렌더러의 서식 적용 방식을 보강할 수 있습니다.

단, 실제 기관 HWPX 양식 원본, 실제 공문 원문, 폰트 파일은 저장소에 추가하지 않습니다.
