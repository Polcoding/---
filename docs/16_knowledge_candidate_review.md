# Knowledge 후보 검토 기준

## 목적

향후 Custom GPT Knowledge 또는 별도 지식 자료로 사용할 수 있는 후보를 검토하기 위한 기준을 정리합니다.

현재 단계에서는 Knowledge 업로드를 하지 않습니다.

## 후보가 될 수 있는 자료

- 비식별화된 문체 샘플
- 원문이 아닌 문서 구조 설명
- 공개 가능한 작성요령
- placeholder 기반 템플릿
- 보안 검토를 통과한 패턴 문서

## 후보에서 제외할 자료

- 실제 공문 원문
- 실제 문서번호가 포함된 자료
- 실제 기관명, 관서명, 담당자명이 포함된 자료
- 민원인, 직원, 업체 등 식별정보가 포함된 자료
- 내부망 자료
- 대외비 또는 비공개 자료
- 실제 업체 견적자료
- 감사ㆍ징계ㆍ수사 등 민감 업무 세부자료

## 검토 절차

1. 자료 유형을 확인합니다.
2. 실제 원문 포함 여부를 확인합니다.
3. 개인정보와 내부 운영정보 포함 여부를 확인합니다.
4. 비식별화 수준을 확인합니다.
5. 문체ㆍ구조ㆍ서식 패턴만 남았는지 확인합니다.
6. 보안 담당 검토 필요 여부를 표시합니다.
7. Knowledge 후보 여부를 결정합니다.

## 판정 값

- `후보 가능`
- `비식별 보완 필요`
- `내부 검토 필요`
- `제외`
