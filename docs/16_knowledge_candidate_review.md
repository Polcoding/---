# Knowledge 후보 검토

## 목적

Custom GPT Knowledge 후보 자료로 사용할 수 있는지 검토하기 위한 기준을 정리합니다.

현재 단계에서는 Knowledge 업로드를 하지 않습니다. Knowledge 업로드는 3단계 완료 후 별도 검토 대상으로만 다룹니다.

## 업로드 가능 후보

다음 조건을 모두 만족하는 경우에만 후보로 볼 수 있습니다.

- 실제 원문이 아님
- 개인정보가 없음
- 실제 기관명, 관서명, 업체명이 없음
- 실제 차량번호, 민원번호, 접수번호가 없음
- 내부 운영정보가 없음
- 대외비 또는 비공개 자료가 아님
- 문체ㆍ구조ㆍ서식 패턴만 포함
- placeholder 기반으로 작성됨
- 보안 검토 체크리스트를 통과함

## 업로드 금지 자료

- 기존 공문 원문
- 실제 문서번호가 포함된 자료
- 실제 개인정보가 포함된 자료
- 실제 기관정보 또는 관서정보가 포함된 자료
- 실제 내부 운영정보가 포함된 자료
- 대외비 또는 비공개 자료
- 내부망 자료
- 실제 민원자료
- 실제 업체 견적자료
- 감사ㆍ징계ㆍ수사ㆍ보안 관련 세부자료

## Knowledge 업로드 전 최종 체크리스트

- [ ] 원문이 아니라 비식별 샘플임
- [ ] 실제 개인정보 없음
- [ ] 실제 기관명ㆍ관서명 없음
- [ ] 실제 업체명 없음
- [ ] 실제 차량번호 없음
- [ ] 실제 민원번호ㆍ접수번호 없음
- [ ] 실제 내부 운영정보 없음
- [ ] 대외비ㆍ비공개 자료 아님
- [ ] 문체ㆍ구조ㆍ서식 패턴만 포함
- [ ] 보안 담당 또는 내부 기준 검토 필요 여부 확인
- [ ] 업로드 목적과 범위가 명확함

## 판정

| 판정 | 의미 |
|---|---|
| 후보 가능 | 모든 체크리스트 통과 |
| 비식별 보완 필요 | 일부 placeholder 치환 필요 |
| 내부 검토 필요 | 보안 기준 확인 전까지 보류 |
| 제외 | Knowledge 후보로 사용하지 않음 |
