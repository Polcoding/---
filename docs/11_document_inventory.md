# 문서 인벤토리

## 목적

향후 문서 패턴을 정리하기 전에 어떤 종류의 문서를 검토 대상으로 삼을지 목록화합니다. 이 문서는 실제 원문을 저장하지 않고, 문서 유형과 검토 상태만 관리합니다.

## 원칙

- 실제 공문 원문을 이 저장소에 보관하지 않습니다.
- 실제 문서번호, 기관명, 담당자명, 수신처, 날짜를 기록하지 않습니다.
- 문서 유형, 용도, 보안 등급, 비식별 필요 여부만 기록합니다.
- 문체ㆍ구조ㆍ서식 패턴 추출 가능성만 검토합니다.

## 문서 유형 예시

- 공문
- 조사표
- 기획서
- 검토보고서
- 업체 연락 메일
- 회의자료
- 작성요령

## 검토 결과 기록 방식

각 문서는 다음 중 하나로 표시합니다.

- `검토 전`
- `비식별 필요`
- `패턴 추출 가능`
- `패턴 추출 제한`
- `제외`

## 제외 기준

다음에 해당하면 패턴 추출 대상에서 제외합니다.

- 개인정보가 핵심 내용에 포함됨
- 내부 운영정보가 구체적으로 포함됨
- 대외비 또는 비공개 자료에 해당함
- 특정 사건이나 민원 세부내용을 식별할 수 있음
- 문체ㆍ구조 추출보다 보안 위험이 큼
