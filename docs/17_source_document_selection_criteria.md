\# 기존 문서 후보 선정 기준



\## 목적



이 문서는 기존 공문, 계획서, 보고서, 조사표, 업체 연락문 등에서 비식별 문체 샘플과 문서 구조를 추출하기 위해 어떤 문서를 우선 후보로 삼을지 정리한 기준입니다.



본 문서는 실제 문서 원문, 실제 파일명, 실제 기관명, 실제 담당자명, 실제 민감정보를 기록하지 않습니다.



\## 기본 원칙



\- 실제 원문은 외부 AI에 입력하지 않습니다.

\- 실제 파일명 대신 TODO 코드 또는 DOC 코드를 사용합니다.

\- 문서 내용 요약이 아니라 문서 유형, 구조, 문체 패턴, 서식 참고 가능성만 확인합니다.

\- 개인정보, 민감정보, 내부 운영정보가 포함될 가능성이 높은 문서는 초기 후보에서 제외합니다.

\- 우선 A등급 또는 B등급 후보만 비식별 샘플 작성 대상으로 검토합니다.



\## 전체 후보 선정 기준



기존 문서 후보는 다음 기준을 우선 적용하여 선정합니다.



| 기준 | 설명 |

|---|---|

| 반복성 | 여러 업무에서 반복적으로 사용할 수 있는 문체와 구조가 있는가 |

| 일반성 | 특정 사건이나 민감 사안이 아니라 일반 행정업무에 가까운가 |

| 비식별 가능성 | 실명, 기관명, 문서번호 등을 제거해도 문체 샘플로 활용 가능한가 |

| 서식 참고성 | 번호 체계, 붙임 표기, 표 구조, 목차 구조를 참고할 수 있는가 |

| 민감도 | 개인정보, 내부 운영정보, 대외비, 감사ㆍ징계ㆍ인사ㆍ수사ㆍ보안 정보가 적은가 |

| 실무성 | 실제 공문, 계획서, 메일 초안 작성에 도움이 되는가 |



\## 우선 제외 기준



다음에 해당하는 문서는 3단계 초기 후보에서 제외합니다.



| 제외 대상 | 제외 이유 |

|---|---|

| 개인정보가 포함된 민원자료 | 실명, 연락처, 주소, 민원번호 등 식별정보 포함 가능성 |

| 감사ㆍ징계ㆍ인사 관련 문서 | 민감정보 및 내부 판단정보 포함 가능성 |

| 수사ㆍ보안 관련 문서 | 외부 처리 부적절 가능성 |

| 실제 관서별 장비ㆍ차량ㆍ운영현황 | 내부 운영정보 포함 가능성 |

| 업체 견적 비교표 또는 계약 검토자료 | 업체 정보, 가격정보, 내부 검토의견 포함 가능성 |

| 대외비 또는 비공개 표시 문서 | 외부 활용 부적절 |

| 내부망 경로, 시스템명, 계정정보 포함 문서 | 보안 위험 |

| 민원인 원문, 진술서, 상담기록 | 개인정보 및 민감 맥락 포함 가능성 |

\## 후보 선정 우선순위



실제 문서를 확인할 때는 다음 순서로 우선 검토합니다.



1\. 공개 가능하거나 민감정보가 거의 없는 문서

2\. 정기적으로 반복되는 일반 행정문서

3\. 문체와 구조가 안정적인 문서

4\. 서식과 번호 체계가 잘 정리된 문서

5\. 특정 사건성이나 개인식별 가능성이 낮은 문서

6\. 비식별 처리 후 문체 샘플로 전환 가능한 문서



초기 후보에서 제외할 문서는 다음과 같습니다.



1\. 개인정보가 중심인 문서

2\. 민원인 원문 또는 민원 세부내역이 포함된 문서

3\. 실제 관서별 운영현황이나 장비현황이 포함된 문서

4\. 업체 견적 금액, 계약 조건, 평가 의견이 포함된 문서

5\. 대외비, 비공개, 감사, 징계, 인사, 수사, 보안 관련 문서
