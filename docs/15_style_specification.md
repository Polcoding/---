# 문체ㆍ서식 명세

## 목적

문서 유형별 폰트, 글씨크기, 줄간격, 자간, 표 양식, 붙임 표기, 번호 체계 등 서식 명세를 정리하기 위한 틀입니다.

## 핵심 원칙

AI는 문서의 의미 구조와 초안을 작성합니다. 폰트, 글씨크기, 줄간격, 자간, 여백, 표 양식 같은 세부 서식은 AI가 직접 맞추는 것이 아니라 향후 템플릿 엔진이 처리해야 합니다.

## 번호 체계

```text
1. / 2. / 3. / 4.
→ 가. / 나. / 다.
→ 1) / 2) / 3)
```

## 붙임 표기

```text
붙임  1. [문서명] 1부.
      2. [문서명] 1부.  끝.
```

## 관리 방식

- 실제 기관의 서식 기준은 `[확인 필요]`로 둡니다.
- 원문을 복사하지 않고 서식 항목명만 정리합니다.
- 향후 템플릿 엔진 구현 시 이 명세를 기준으로 별도 매핑합니다.
