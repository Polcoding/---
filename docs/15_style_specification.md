# 문서 서식 명세표

## 목적

이 문서는 공공기관 행정문서의 폰트, 글씨 크기, 자간, 줄간격, 표 양식, 붙임 표기, 번호 체계 등을 정리하기 위한 서식 명세표입니다.

AI가 문서 서식을 직접 맞추는 것이 아니라, 향후 HWPX/XLSX 템플릿 엔진이 확인된 서식 기준을 적용한다는 원칙을 따릅니다.

실제 기관 서식 값은 확인 전까지 [확인 필요]로 표시하며, 실제 공문 원문이나 실제 기관 서식 값을 임의로 만들지 않습니다.

## 기본 원칙

- 실제 공문 원문을 사용하지 않습니다.
- 실제 서식 값을 임의로 만들지 않습니다.
- 확인되지 않은 폰트, 크기, 자간, 줄간격, 여백은 [확인 필요]로 표시합니다.
- AI는 내용 구조와 문장 초안을 생성합니다.
- 템플릿 엔진은 서식 적용을 담당합니다.
- HWPX와 XLSX는 별도 템플릿 기준으로 관리합니다.
- 이번 단계에서는 HWPX/XLSX 생성 코드를 만들지 않습니다.
- Make.com, API, 이메일 자동화 구현은 하지 않습니다.

## AI와 템플릿 엔진 역할 분리

| 구분 | AI 담당 | 템플릿 엔진 담당 |
|---|---|---|
| 공문 제목 | 제목 문구 생성 | 글꼴, 크기, 정렬 적용 |
| 본문 | 문장과 번호 구조 생성 | 줄간격, 자간, 들여쓰기 적용 |
| 표 | 표에 들어갈 항목 제안 | 셀 병합, 테두리, 열 너비 적용 |
| 붙임 | 붙임 목록 생성 | 붙임 정렬과 끝. 표기 적용 |
| 조사표 | 항목과 작성요령 생성 | XLSX 셀 서식, 테두리, 너비 적용 |

## 공통 번호 체계

```text
1.
가.
1)
가)
```

상위 번호가 1., 2., 3. 형식이면 다음 상위 번호도 4. 형식을 유지합니다.

3), 4)처럼 상위 번호 체계를 잘못 사용하지 않습니다.

## 붙임 표기 기준

```text
붙임  1. [문서명] 1부.
      2. [문서명] 1부.  끝.
```
