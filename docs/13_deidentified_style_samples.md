# 비식별 문체 샘플

## 목적

실제 사례 없이 placeholder 기반으로 공공기관 행정문서의 문체ㆍ구조ㆍ표현 패턴을 정리합니다.

## 작성 원칙

- 실제 공문 원문을 붙여넣지 않습니다.
- 실제 기관명, 관서명, 업체명, 사람 이름을 만들지 않습니다.
- 실제 차량번호, 민원번호, 접수번호, 내부 운영정보를 만들지 않습니다.
- 모든 예시는 `[기관 A]`, `[문서 A]`, `[업무명 확인 필요]` 같은 placeholder만 사용합니다.

## 공문

```text
[업무 발생 배경]에 따라 [대상 기관]의 [확인 대상] 현황을 확인하고, 향후 업무 추진을 위한 기초자료로 활용하고자 합니다.
```

```text
이에 각 대상 기관에서는 아래 사항을 확인하여 [제출기한 확인 필요]까지 제출하여 주시기 바랍니다.
```

## 추진계획서

```text
[업무명 확인 필요] 추진을 통해 [대상]의 [개선 목적]을 달성하고, 관련 업무의 효율적인 운영 기반을 마련하고자 합니다.
```

```text
세부 추진일정, 소요예산, 참여 대상은 내부 검토 후 확정하며, 현재 단계에서는 `[확인 필요]`로 표시합니다.
```

## 결과보고서

```text
[업무명 확인 필요] 추진 결과, [주요 결과 요약]을 확인하였으며 향후 개선사항은 [후속 조치 확인 필요]에 따라 검토할 예정입니다.
```

```text
본 결과는 [활용 목적 확인 필요]을 위한 참고자료이며, 세부 수치와 근거자료는 내부 기준에 따라 별도 확인이 필요합니다.
```

## 현황조사 공문

```text
[조사 목적]을 위하여 [대상 기관]의 [조사 항목] 현황을 파악하고자 하오니, 붙임 양식에 따라 작성하여 제출하여 주시기 바랍니다.
```

```text
외부 AI 처리용 초안에는 실제 관서명, 차량번호, 세부 수량, 내부 운영정보를 포함하지 않으며, 내부 실무 적용 양식의 기재 범위는 `[내부 기준에 따라 확인 필요]`로 표시합니다.
```

## 업체 메일

```text
제목: [업무명 확인 필요] 관련 자료/견적 요청의 건
```

```text
본 요청은 사전 검토를 위한 자료 요청이며, 계약 또는 업체 선정이 확정된 사항은 아닙니다.
```

```text
회신 시 개인정보, 참석자 명단, 내부자료 또는 민감정보가 포함된 자료는 제외하여 주시기 바랍니다.
```
