# 비식별 문체 샘플 세트

## 목적

이 문서는 기존 문서 원문을 AI에 학습시키지 않고, 문체ㆍ구조ㆍ서식 패턴만 비식별화하여 관리하기 위한 샘플 누적 문서입니다.

여기에 작성하는 내용은 실제 원문, 실제 파일명, 실제 문서명, 실제 기관명, 실제 담당자명, 실제 업체명, 실제 날짜, 실제 문서번호, 실제 민원번호, 실제 차량번호, 실제 장비명, 실제 수량, 실제 견적 금액을 포함하지 않습니다.

## 작성 원칙

- 원문 문장 복사 금지
- 원문 요약 금지
- 실제 파일명 기록 금지
- 실제 기관명, 관서명, 업체명 기록 금지
- 실제 개인정보, 내부 운영정보 기록 금지
- placeholder 중심 작성
- A/B등급 후보만 샘플 작성 가능
- C등급은 내부 참고만 가능하므로 샘플 후보에서 제외
- D등급은 외부 처리 금지로 분류하고 샘플 작성 대상에서 제외
- Custom GPT Knowledge 업로드는 아직 하지 않음

## SAMPLE-001. 조사 지시 공문 문체 샘플

문서 유형:
조사 지시 공문

보안등급:
B등급 후보

원문 사용 여부:
원문 미사용. 문장 구조만 일반화한 예시.

비식별 문체 샘플:
[업무 발생 배경]에 따라 [대상 기관/부서]의 [조사 대상] 현황을 확인하고, 향후 [활용 목적]을 위한 기초자료로 활용하고자 합니다.

각 [대상 기관/부서]에서는 [조사 기준일] 현재의 [조사 대상] 현황을 붙임 서식에 따라 작성하여 [제출기한]까지 제출하여 주시기 바랍니다.

주의사항:
실제 기관명, 실제 차량번호, 실제 장비명, 실제 수량, 실제 담당자명은 포함하지 않습니다.

검수 상태:
초안
