# PoC 구현 준비 체크리스트

## 문서 준비

- [ ] AI 출력 JSON 스키마가 있는가
- [ ] XLSX PoC 설계 문서가 있는가
- [ ] HWPX PoC 설계 문서가 있는가
- [ ] 렌더러 요구사항 문서가 있는가
- [ ] 샘플 JSON이 있는가
- [ ] 보안등급 기준이 있는가
- [ ] 서식 명세표가 있는가

## 샘플 JSON 준비

- [ ] 공문 샘플 JSON이 있는가
- [ ] 추진계획서 샘플 JSON이 있는가
- [ ] 업체 메일 샘플 JSON이 있는가
- [ ] 조사표 샘플 JSON이 있는가
- [ ] 보안 차단 샘플 JSON이 있는가
- [ ] 모든 샘플이 JSON 문법 검증을 통과했는가
- [ ] 모든 샘플이 placeholder 기반인가

## 보안 준비

- [ ] 실제 원문이 없는가
- [ ] 실제 개인정보가 없는가
- [ ] 실제 기관명이나 업체명이 없는가
- [ ] 실제 차량번호나 장비현황이 없는가
- [ ] 실제 금액이나 계약 조건이 없는가
- [ ] API 키나 비밀번호가 없는가
- [ ] 자동 발송이 없는가

## 구현 시작 조건

- [ ] 사용자가 명시적으로 PoC 구현 시작을 지시했는가
- [ ] 구현 대상이 XLSX 렌더러 PoC로 제한되어 있는가
- [ ] 실제 업무자료 없이 샘플 JSON만 사용하는가
- [ ] 생성 파일은 로컬 샘플 산출물로 제한되는가

## Step 14 완료 체크리스트

- [x] PoC 구현 준비 체크리스트가 작성되었는가
- [x] 문서 간 일관성 점검 항목이 작성되었는가
- [x] 구현 전 보안 게이트가 작성되었는가
- [x] 구현 우선순위가 작성되었는가
- [x] 실제 구현 코드를 작성하지 않았는가
- [x] 실제 원문이나 개인정보를 추가하지 않았는가
