# Phase 3 저장소 밖 HWPX 취급 및 수동 preview 체크리스트

## 목적

`docs/112_phase3_external_hwpx_and_manual_preview_criteria.md` 기준으로 저장소 밖 HWPX 양식 취급과 사용자 수동 preview 기준이 충분히 구체화되었는지 확인합니다.

## 저장소 밖 HWPX 취급 확인

| 완료 | 항목 |
|---|---|
| [x] | 실제 원본을 저장소 안에 복사하지 않는 기준이 있는가 |
| [x] | 저장소 밖 로컬 작업 폴더 기준이 있는가 |
| [x] | 파일명에 실제 기관명, 부서명, 문서번호를 쓰지 않는 기준이 있는가 |
| [x] | A/B/C/D 등급 판단 기준이 있는가 |
| [x] | C/D등급 중단 기준이 있는가 |
| [x] | 식별 요소 제거 기준이 있는가 |
| [x] | 파일 속성, 미리보기, 마지막 저장자 정보 확인 기준이 있는가 |
