# HWPX placeholder 템플릿 검수 체크리스트

## placeholder 검수

- [ ] 모든 동적 값이 placeholder로 처리되었는가
- [ ] 실제 문서번호가 없는가
- [ ] 실제 시행일자가 없는가
- [ ] 실제 기관명, 수신처, 담당자명이 없는가
- [ ] 실제 개인정보가 없는가

## 서식 검수

- [ ] 제목 영역이 분리되어 있는가
- [ ] 본문 영역이 분리되어 있는가
- [ ] 붙임 영역이 분리되어 있는가
- [ ] 표 영역이 분리되어 있는가
- [ ] 번호체계가 유지되는가
- [ ] 붙임 표기 기준이 반영되었는가

## 문체 검수

- [ ] 구어체 표현이 없는가
- [ ] 개조식 표현이 가능한 구조인가
- [ ] 불필요하게 장황한 문장 삽입을 유도하지 않는가
- [ ] [확인 필요] 표시가 가능한 구조인가
- [ ] 공공기관 행정문구를 반영할 수 있는 구조인가

## 보안 검수

- [ ] 실제 원문이 없는가
- [ ] 실제 민원정보가 없는가
- [ ] 실제 차량번호, 장비명, 수량이 없는가
- [ ] 실제 내부 운영정보가 없는가
- [ ] Git에 실제 HWPX 원본을 추가하지 않았는가

## Step 22 완료 체크리스트

- [x] HWPX placeholder 전략 문서가 작성되었는가
- [x] HWPX template manifest가 작성되었는가
- [x] HWPX placeholder 템플릿 검수 체크리스트가 작성되었는가
- [x] 실제 HWPX 파일을 생성하지 않았는가
- [x] 실제 기관 양식을 사용하지 않았는가
- [x] 실제 원문이나 개인정보를 추가하지 않았는가
- [x] 공문체와 서식 충실도 기준이 반영되었는가
