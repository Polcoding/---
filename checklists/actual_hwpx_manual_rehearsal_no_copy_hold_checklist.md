# 실제 양식 수동 리허설 복사본 없음 보류 체크리스트

## 상태 기록 확인

| 완료 | 항목 |
|---|---|
| [x] | 사용자가 비식별 HWPX 작업 복사본이 없다고 밝힌 상태를 기록했는가 |
| [x] | 복사본 없음 상태에서 한컴 preview 결과를 임의로 기록하지 않았는가 |
| [x] | `문제 없음` 결과를 실제 확인 없이 만들지 않았는가 |
| [x] | 실제 파일명, 실제 경로, 실제 기관명을 기록하지 않았는가 |

## 보류 판단 확인

| 완료 | 항목 |
|---|---|
| [x] | 실제 양식 수동 리허설을 보류한다고 정리했는가 |
| [x] | HWPX output을 재생성하지 않았는가 |
| [x] | renderer 코드를 변경하지 않았는가 |
| [x] | normalizer 코드를 변경하지 않았는가 |
| [x] | fixture JSON을 추가하거나 변경하지 않았는가 |
| [x] | HWPX payload 구조를 변경하지 않았는가 |

| 완료 | 항목 |
|---|---|

| [x] | 실제 공문 원문 또는 보고서 원문을 추가하지 않았는가 |
