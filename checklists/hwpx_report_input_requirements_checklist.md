# HWPX 보고서 입력 요구사항 체크리스트

## 공통 입력

- [ ] `document_type`이 4종 중 하나로 지정되었는가
- [ ] `output_targets`에 `hwpx`가 포함되어 있는가
- [ ] `security_review`가 있는가
- [ ] `missing_fields`가 있는가
- [ ] `human_review_required`가 `true`인가
- [ ] 제목이 없을 경우 문서 유형별 placeholder 제목을 사용하는가

## 보안 입력

- [ ] 실제 원문이 포함되지 않았는가
- [ ] 실제 개인정보가 포함되지 않았는가
- [ ] 실제 기관명, 문서번호, 담당자명이 포함되지 않았는가
- [ ] 실제 차량번호, 장비명, 내부 운영정보가 포함되지 않았는가
- [ ] C/D등급 문서를 렌더링 대상으로 삼지 않았는가

## 누락값 정책

- [ ] 예산 누락 시 임의 생성하지 않는가
- [ ] 일정 누락 시 임의 생성하지 않는가
- [ ] 실적 수치 누락 시 임의 생성하지 않는가
- [ ] 담당자 누락 시 임의 생성하지 않는가
- [ ] 수량 누락 시 임의 생성하지 않는가
- [ ] 누락값이 `missing_fields` 또는 `[확인 필요]`로 표시되는가

## 문서 유형별 확인

- [ ] 원페이지 보고서 필수 필드가 확인되었는가
- [ ] 추진계획서 필수 필드가 확인되었는가
- [ ] 결과보고서 필수 필드가 확인되었는가
- [ ] 검토보고서 필수 필드가 확인되었는가

## 렌더링 힌트

- [ ] 긴 객체/배열 필드는 HWPX에서 축약할지 결정했는가
- [ ] 표 구조가 필요한 필드는 별도 단계로 분리했는가
- [ ] 검토용 메타정보를 실제 본문에 노출할지 결정했는가
- [ ] output은 사람 검토용 초안임을 유지하는가
