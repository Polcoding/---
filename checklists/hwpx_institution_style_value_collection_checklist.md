# HWPX 기관 표준 서식값 수집 체크리스트

## 목적

기관 표준 서식값을 수집하기 전 확인할 항목을 정리합니다.

최신 `[확인 필요]` 유지 기준은 `docs/126_style_profile_confirmation_value_collection_criteria.md`를 따릅니다. 이 체크리스트는 실제 서식값을 확정하는 문서가 아니며, 확인되지 않은 값은 계속 `[확인 필요]`로 둡니다.

## 수집 전 중단 조건

- [ ] GitHub Desktop Changes에 실제 HWP/HWPX 파일이 나타나지 않는가
- [ ] 실제 기관 HWPX/HWP 원본을 저장소 밖에서만 다루는가
- [ ] 실제 공문 원문, 결재선, 직인, 로고, 수신자 정보가 수집 기록에 들어가지 않는가
- [ ] 실제 기관명, 실제 부서명, 실제 문서번호, 실제 파일명을 기록하지 않는가
- [ ] 폰트 파일을 저장소에 추가하지 않는가
- [ ] 확인되지 않은 값은 `[확인 필요]`로 유지하는가
