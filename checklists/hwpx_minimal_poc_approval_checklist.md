# HWPX 최소 PoC 구현 승인 체크리스트

## 구현 전 조건

- [ ] HWPX placeholder 템플릿 전략이 확정되었는가
- [ ] HWPX template manifest가 작성되었는가
- [ ] HWPX placeholder 템플릿 검수 체크리스트가 있는가
- [ ] HWPX 서식 충실도 체크리스트가 있는가
- [ ] 공공기관 문체 체크리스트가 있는가
- [ ] 입력 샘플 JSON이 placeholder 기반인가
- [ ] 실제 기관 양식을 사용하지 않는가
- [ ] 실제 공문 원문을 사용하지 않는가
- [ ] 실제 개인정보를 사용하지 않는가
- [ ] output 산출물 Git 제외 계획이 있는가

## 구현 범위 승인

- [ ] 공문 최소 PoC 범위가 확정되었는가
- [ ] 추진계획서 최소 PoC 범위가 확정되었는가
- [ ] 실제 발송용 문서가 아니라 검토용 초안으로 제한되었는가
- [ ] high 또는 blocked 입력 차단 조건이 확정되었는가
- [ ] 사용자가 명시적으로 HWPX 최소 PoC 구현 시작을 승인했는가

## 금지 확인

- [ ] 실제 기관 HWPX 양식 원본을 사용하지 않는가
- [ ] 실제 직인, 로고, 결재선 정보를 사용하지 않는가
- [ ] 실제 민원자료, 장비현황, 차량현황을 사용하지 않는가
- [ ] API, 이메일 자동화, Make.com 연동을 하지 않는가
