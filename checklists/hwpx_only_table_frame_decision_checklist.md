# HWPX 일원화와 표 틀 우선 결정 체크리스트

## 결정 범위 확인

| 완료 | 항목 |
|---|---|
| [x] | HWPX를 최종 보고서 중심 형식으로 유지했는가 |
| [x] | DOCX/Word 변환 흐름을 공식 흐름으로 채택하지 않았는가 |
| [x] | HWPX 표를 현재 단계에서는 표 틀과 배치 검수 대상으로 제한했는가 |
| [x] | 표 내부 데이터 자동화를 향후 Excel/한셀 연동 후보로 분리했는가 |
| [x] | XLSX 조사표를 HWPX 보고서보다 우선하는 것으로 바꾸지 않았는가 |

## 사용자 확인 범위 확인

| 완료 | 항목 |
|---|---|
| [x] | 사용자가 확인할 표 항목을 위치, 폭, 줄바꿈, 겹침, 여백으로 제한했는가 |
| [x] | 표 안 실제 값, 수량, 금액, 대상 목록을 기록하지 않도록 했는가 |
| [x] | `table_scope: frame_only`로 문제 없음 결과를 전달할 수 있게 했는가 |
| [x] | gap이 있을 때 실제 표 값 없이 증상만 기록하도록 했는가 |
