# HWPX 서식 충실도 체크리스트

## 기본 서식

- [ ] 제목 글꼴이 기준과 일치하는가
- [ ] 제목 크기가 기준과 일치하는가
- [ ] 본문 글꼴이 기준과 일치하는가
- [ ] 본문 크기가 기준과 일치하는가
- [ ] 자간이 기준과 일치하는가
- [ ] 줄간격이 기준과 일치하는가
- [ ] 문단 간격이 기준과 일치하는가
- [ ] 여백이 기준과 일치하는가

## 번호체계

- [ ] 상위 번호가 1. / 2. / 3. 형식으로 유지되는가
- [ ] 하위 번호가 가. / 나. / 다. 형식으로 유지되는가
- [ ] 그 아래 번호가 1) / 2) / 3) 형식으로 유지되는가
- [ ] 3), 4) 같은 잘못된 상위 번호가 없는가

## 표 양식

- [ ] 표 테두리가 유지되는가
- [ ] 셀 병합이 깨지지 않았는가
- [ ] 셀 여백이 유지되는가
- [ ] 표 머리글이 구분되는가

## 붙임

- [ ] 붙임 표기가 기준 형식과 맞는가
- [ ] 끝. 위치가 올바른가
- [ ] 붙임 번호가 순서대로 표시되는가

## 보안

- [ ] 실제 개인정보가 없는가
- [ ] 실제 기관명, 관서명, 업체명이 없는가
- [ ] 실제 문서번호, 민원번호, 차량번호가 없는가
- [ ] 실제 장비명, 수량, 내부 운영정보가 없는가
