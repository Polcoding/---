# HWPX 보고서와 Excel/한셀 표 데이터 역할 분리 체크리스트

## 역할 분리 확인

| 완료 | 항목 |
|---|---|
| [x] | HWPX를 최종 보고서 형식으로 유지했는가 |
| [x] | Excel/한셀을 표 데이터 보조 후보로만 분리했는가 |
| [x] | XLSX 또는 한셀을 HWPX보다 우선하는 것으로 바꾸지 않았는가 |
| [x] | 표 틀과 표 내부 데이터의 역할을 분리했는가 |

## 문서유형별 범위 확인

| 완료 | 항목 |
|---|---|
| [x] | `one_page_report`의 표 데이터 후보를 실제값 없이 정리했는가 |
| [x] | `project_plan`의 일정표, 예산표, 대상 목록 후보를 실제값 없이 정리했는가 |
| [x] | `result_report`의 실적 집계와 예산 집행 후보를 실제값 없이 정리했는가 |
| [x] | `review_report`의 위험요소 매트릭스 후보를 확정 판단 없이 정리했는가 |
