# 원천 문서 검토 체크리스트

## 원칙

- [ ] 실제 원문을 저장소에 복사하지 않음
- [ ] 문서 유형만 기록함
- [ ] 실제 문서번호를 기록하지 않음
- [ ] 실제 기관명ㆍ관서명을 기록하지 않음
- [ ] 실제 담당자명 또는 연락처를 기록하지 않음
- [ ] 실제 민원자료를 기록하지 않음
- [ ] 실제 업체 견적자료를 기록하지 않음

## 검토 항목

- [ ] 문서 유형 확인
- [ ] 보안 분류 확인
- [ ] 비식별 필요 여부 확인
- [ ] 문체ㆍ구조 추출 가능 여부 확인
- [ ] 제외 대상 여부 확인
- [ ] 내부 승인 필요 여부 확인

## 판정

- [ ] 패턴 추출 가능
- [ ] 비식별 보완 필요
- [ ] 내부 검토 필요
- [ ] 제외
