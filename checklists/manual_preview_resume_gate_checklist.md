# 수동 Preview 재개 게이트 체크리스트

## 목적

`docs/150_manual_preview_resume_gate.md` 기준으로 실제 HWPX 수동 preview를 재개해도 되는지 확인합니다.

이 체크리스트는 실제 HWPX 파일을 Git에 추가하거나 실제 업무값을 기록하기 위한 양식이 아닙니다.

## 재개 전 필수 확인

| 완료 | 표시 | 항목 |
|---|---|---|
| [ ] | `[사용자 확인 필요]` | 저장소 밖 비식별 작업 복사본이 준비되었는가 |
| [ ] | `[사용자 확인 필요]` | 실제 원문, 개인정보, 기관명, 문서번호가 제거되었는가 |
| [ ] | `[사용자 확인 필요]` | 실제 표 데이터, 수량, 금액, 대상 목록이 제거되었는가 |
| [ ] | `[사용자 확인 필요]` | GitHub Desktop Changes에 실제 HWPX/HWP 원본이 보이지 않는가 |
| [ ] | `[사용자 확인 필요]` | 사용자가 한컴에서 파일을 열어볼 수 있는가 |
| [ ] | `[사용자 확인 필요]` | 검수 범위가 레이아웃 확인으로 제한되는가 |
| [ ] | `[Codex 처리 가능]` | 외부 HWPX 자동 채우기 자료가 구조 분석 참고용으로만 연결되는가 |
