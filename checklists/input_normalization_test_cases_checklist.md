# 입력 정규화 테스트 케이스 체크리스트

## 목적

`docs/63_input_normalization_test_cases.md`가 Phase 2 보안 필터와 입력 정규화 스키마에 맞게 작성되었는지 확인합니다.

## 기준 문서

- `docs/61_input_normalization_schema.md`
- `docs/62_security_filter_requirements.md`
- `docs/60_hwpx_report_input_requirements.md`

| [x] | 보안 검토 입력 케이스가 포함되었는가 |

| [x] | 각 케이스에 예상 `routing_decision`이 포함되었는가 |

|---|---|
| [x] | 실제 개인정보를 포함하지 않았는가 |
