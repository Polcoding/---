# Phase 2 사용자 입력 및 수동 운영 체크리스트

## 목적

`docs/83_phase2_user_input_and_manual_operation_checkpoints.md` 기준으로 입력 전, HWPX 생성 전, 생성 후 검수 항목이 정리되었는지 확인합니다.

## 입력 전 체크

| 완료 | 항목 |
|---|---|
| [x] | 실제 공문 원문 제외 기준이 있는가 |
| [x] | 실제 개인정보 제외 기준이 있는가 |
| [x] | 실제 기관명ㆍ부서명 제외 기준이 있는가 |
| [x] | 실제 문서번호ㆍ민원번호ㆍ사건번호 제외 기준이 있는가 |
| [x] | 실제 예산ㆍ계약값 제외 기준이 있는가 |
| [x] | 실제 내부 운영정보 제외 기준이 있는가 |
| [x] | 자동 발송ㆍ결재ㆍ계약ㆍ예산 집행 금지 기준이 있는가 |
| [x] | 확인 필요값을 `[확인 필요]`로 남기는 기준이 있는가 |
