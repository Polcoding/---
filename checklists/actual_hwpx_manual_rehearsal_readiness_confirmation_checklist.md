| 완료 | 항목 |
|---|---|
| [x] | 사용자가 실제 원본과 작업 복사본이 저장소 밖에만 있음을 확인했는가 |
| [x] | 사용자가 실제 본문, 기관명, 문서번호, 결재선, 직인, 로고, 담당자 정보 제거를 확인했는가 |
