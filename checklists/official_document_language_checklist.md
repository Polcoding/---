# 공공기관 문체 체크리스트

## 문체

- [ ] 구어체 표현이 없는가
- [ ] 불필요하게 장황한 설명이 없는가
- [ ] 개조식 또는 명사형 표현이 적절히 사용되었는가
- [ ] 확정되지 않은 내용이 단정되지 않았는가
- [ ] [확인 필요] 표시가 필요한 곳에 들어갔는가

## 행정문구

- [ ] 제출 요청 문구가 공문체로 작성되었는가
- [ ] 협조 요청 문구가 공문체로 작성되었는가
- [ ] 검토 필요 문구가 포함되었는가
- [ ] 개인정보 제외 안내 문구가 포함되었는가
- [ ] 실제 발송 전 내부 검토 문구가 포함되었는가

## 금지 표현

- [ ] 계약 확정 표현이 없는가
- [ ] 업체 선정 확정 표현이 없는가
- [ ] 예산 확보 단정 표현이 없는가
- [ ] 법률적으로 문제없다는 단정 표현이 없는가
- [ ] 민원인 개인정보를 외부 전달하는 표현이 없는가
