# 1단계 완료 체크리스트

## 목적

공공기관 행정문서 AI 자동화 시스템 1단계 준비가 완료되었는지 점검합니다.

## 체크 항목

| 완료 | 항목 |
|---|---|
| [ ] | Custom GPT Instructions 작성 완료 |
| [ ] | Conversation Starters 작성 완료 |
| [ ] | Knowledge 업로드 없음 |
| [ ] | Capabilities 초기 비활성 권장 |
| [ ] | Actions/Apps 미사용 |
| [ ] | 테스트 케이스 5개 이상 준비 |
| [ ] | 평가표 준비 |
| [ ] | 보안 정책 문서 작성 |
| [ ] | 비식별화 규칙 작성 |
| [ ] | 위험 요청 예시 작성 |
| [ ] | 실제 민감정보 포함 없음 |
| [ ] | README 작성 완료 |
| [ ] | AGENTS.md 작성 완료 |
| [ ] | 향후 자동화 전 체크리스트 작성 완료 |

## 완료 판단

모든 항목이 확인되어야 1단계 준비가 완료된 것으로 봅니다. 단, 실제 운영 전에는 조직 내부 보안정책과 최신 공식 문서를 별도로 확인해야 합니다.
